Welcome!

OASIS is a web based system designed for large classes. It provides a powerful system for both the practice and assessment of skill based questions.



The primary goal of OASIS is to help you with your study; any suggestions you have for features that would help with this are welcome. Several of the capabilities of OASIS have originated from student suggestions. Fault reports are also important.
