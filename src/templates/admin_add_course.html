Quick Configuration

OASIS allows a lot of functionality in how a course is configured. The following options will set some sensible defaults for different situations. You can fine-tune the configuration later, if needed. Any of the options set here can be changed later, these choices do not limit what's possible.

Course Type

Choose the type of course that best suits your needs, as a starting point:


Demonstration

This is suitable for a demonstration course. Practice questions will be available to all users, by default, and students can sign themselves up to the course to do any assessments.


Casual

Useful for small courses (1-30 students) where a single course administrator can manage the student membership themselves.


Standard

A typical large organization course. Enrolment information comes from an external system, and there may be multiple staff assigned to the course.


Large Scale.

There will be many staff assigned to the course, and several groups of students. Possibly several thousand students in total.

Repetition

Will the course run regularly, or will it only run once, for now?


Eternal

The course is always available, students may come and go. This is especially suitable for demonstration courses.


Single

This course is a once-off. To repeat it in the future, additional configuration will be needed.


Annual

Repeats every year, during a similar time period (semester/term)


Often

This course runs multiple times a year.

Visibility

How visible will the course content be, by default?

Note that visibility is controllable to a finer level from within the course itself, this setting just controls the default.


Open

Anyone with an account can practice the questions in the course. This can allow students to try a course out before they enrol in it. Also appropriate for demonstration courses.


Refresher

Students who are, or have been, enrolled in the course in the past can access the practice section. This allows them to go back and refresh their skills later.


Closed

Only students with a current active enrolment in the course can access the practice section.


Hidden

The course does not show up in the practice section at all. This is useful if you're just using it to learn how to use OASIS and experimenting. You can open it up later.
