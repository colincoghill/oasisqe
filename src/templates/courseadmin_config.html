Configuration

Course Name

Please note that if linked to external systems, such as university enrolment databases, the course "name" may be used to find information, so be careful when changing it.

Enrolment

With "open" enrolment, students can enrol themselves in the course. This is good for demonstrations or non academic related content.

Course Visibility

Who can access the course practice content. If set to "All" then people who are not enrolled in the course can practice questions, but will not show up in course statistics. This can be overridden topic by topic, and is a good way to allow students to refresh practice on courses they've taken previously, or try out a course they're considering.
